# kollus-play-video-java
Kollus Play Sample Code for Java

1. Requirement
- A higher than Java JDK 1.6 
- A higher than Maven 3.2.5 

2. How To Run
```
  $ git clone https://github.com/kollus-service/kollus-play-video-java.git
  $ cd kollus-play-video-java
  $ mvn clean install
  $ mvn spring-boot:run
```
